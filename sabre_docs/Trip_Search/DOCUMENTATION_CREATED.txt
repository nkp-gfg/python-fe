YOUR REQUIREMENT vs SOLUTION
═══════════════════════════════════════════════════════════════════════════════

YOUR REQUIREMENT:
  "Get list of passengers with phone numbers for a specific flight by passing
   flight number, flight date and airline code"

THE SOLUTION:
  
  API:           Trip_SearchRQ v4.5.0
  Endpoint:      http://usg.sabre.com/websvc
  Protocol:      SOAP over HTTP/HTTPS
  View:          TripSearchBasic
  Response:      STL v1.19.0 (XML)
  
  REQUEST PARAMETERS:
    • FlightNumber       (e.g., "0001")
    • DepartureDateTime  (e.g., "2016-12-20T07:00:00")
    • AirlineCode        (e.g., "ET")
    • DepartureAirport   (optional but recommended, e.g., "JFK")
  
  RESPONSE DATA:
    • PNR Locator        (e.g., "FZVBDK")
    • Passenger Names    (LastName: "SMITH", FirstName: "JOHN MR")
    • Seat Assignment    (e.g., "12A")
    • Phone Numbers      (e.g., "2125551234" with type "Home")
    • Multiple phones per PNR

QUICK ANSWER TO YOUR QUESTION
═══════════════════════════════════════════════════════════════════════════════

Q: "What API do I need to call to get passengers and phone numbers for a 
    flight by passing flight number, date, and airline code?"

A: Trip_SearchRQ v4.5.0

   Endpoint: http://usg.sabre.com/websvc
   
   Use TripSearchBasic view - it automatically returns:
   • Passenger names
   • Phone numbers
   • PNR locators
   • Seat assignments
   
   Sample Request:
   
   <?xml version="1.0" encoding="UTF-8"?>
   <Trip_SearchRQ Version="4.5.0" xmlns="http://webservices.sabre.com/triprecord">
       <ReadRequests>
           <ReservationReadRequest>
               <FlightCriteria>
                   <Flight>
                       <FlightNumber>0001</FlightNumber>
                       <DepartureDateTime Is="2016-12-20T07:00:00"/>
                       <AirlineCode>ET</AirlineCode>
                       <DepartureAirport>JFK</DepartureAirport>
                   </Flight>
               </FlightCriteria>
               <PosCriteria AirlineCode="ET"/>
               <ReturnOptions ViewName="TripSearchBasic" 
                             ResponseFormat="STL"
                             SearchType="ACTIVE"/>
           </ReservationReadRequest>
       </ReadRequests>
   </Trip_SearchRQ>
   
   Response includes XML with:
   • PNR Locators
   • Passenger FirstName/LastName
   • Phone Numbers (with type: Home/Business/Mobile)
   • Seat Assignments
